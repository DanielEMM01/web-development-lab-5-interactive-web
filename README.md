# Web Development Lab 5 – Creating an Interactive Webpage

This lab is a two page website  
The website is based on a real entertainment company called **D.E.M Entertainment Group**. It is my sole proprietorship company.

## The Project Pages
- index.html – Home page
- contact.html – Contact page

## The Project Structure
- index.html
- contact.html
- styles/styles.css
- images/Green.Ent.png

## The Features according to rubric
- Shared header and footer on both pages
- Navigation links between the home page and contact page
- Home page with two content sections
- Contact page with a form and submit button
- Single external CSS file used for both pages
- Logo image with descriptive alt text
